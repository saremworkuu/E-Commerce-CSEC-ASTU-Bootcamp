# LuxeCart E-Commerce

A simple e-commerce frontend built with React, TypeScript, and Vite.

## Features

- Product listing and product details pages
- Shopping cart with add/remove quantity support
- Basic auth pages (login and register UI)
- Shop, contact, and home pages

## Tech Stack

- React
- TypeScript
- Vite
- React Router

## Project Structure

- `frontend/` - Main React app
- `backend/` - Reserved for backend API (currently empty)

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm

### Run Frontend Locally

1. Go to the frontend folder:
   `cd frontend`
2. Install dependencies:
   `npm install`
3. Start the development server:
   `npm run dev`

The app runs on port `3000` by default.

## Build for Production

From the `frontend` folder:

- `npm run build` - Create production build
- `npm run preview` - Preview production build locally

## Notes

- Backend integration can be added later in the `backend` folder.
